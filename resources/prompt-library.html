Ready-to-use AI prompts to supercharge your productivity across every business function

Lead Qualification Script

Sales

Generate targeted questions to qualify potential leads based on your ideal customer profile.

Prompt:
"Create a lead qualification script for [INDUSTRY] prospects. Include 10 discovery questions that help identify: budget range ($X-Y), decision-making process, timeline, current pain points with [SOLUTION AREA], and technical requirements. Format as a conversational flow with follow-up questions."

Objection Handling Responses

Sales

Craft compelling responses to common sales objections with data-backed arguments.

Prompt:
"Generate professional responses to these common objections for [PRODUCT/SERVICE]: 'Too expensive', 'Not the right time', 'Need to think about it', 'Happy with current solution'. Include statistics, social proof, and risk reversal techniques. Tone: consultative and empathetic."

Content Calendar Generator

Marketing

Create a comprehensive content calendar with topics, formats, and distribution channels.

Prompt:
"Create a 30-day content calendar for [BRAND] targeting [AUDIENCE]. Include: 15 blog topics, 20 social media posts, 5 email campaigns, and 3 video concepts. Focus on [MAIN THEME/PRODUCT]. Include content pillars: educational (40%), promotional (20%), entertaining (20%), user-generated (20%). Format as a table with dates, platforms, and CTAs."

Email Campaign Sequences

Marketing

Design automated email sequences for lead nurturing and customer onboarding.

Prompt:
"Create a 7-email nurture sequence for [PRODUCT] leads who downloaded [LEAD MAGNET]. Include: welcome email, 3 educational emails, 1 social proof email, 1 objection-handling email, 1 final offer email. Each email should be 150-200 words with clear CTAs. Include subject lines and optimal send timing."

Code Review Checklist

Engineering

Generate comprehensive code review guidelines tailored to your tech stack.

Prompt:
"Create a code review checklist for [PROGRAMMING LANGUAGE] projects using [FRAMEWORK]. Include sections for: code quality, security vulnerabilities, performance optimization, testing coverage, documentation, and team conventions. Provide specific examples and automated tools recommendations."

API Documentation Template

Engineering

Create clear, developer-friendly API documentation with examples and use cases.

Prompt:
"Generate comprehensive API documentation for [API NAME]. Include: overview, authentication methods, all endpoints with HTTP methods, request/response examples, error codes, rate limiting, SDKs, and getting started guide. Use OpenAPI 3.0 format with practical code samples in Python, JavaScript, and cURL."

Job Description Generator

HR

Create compelling job descriptions that attract top talent and set clear expectations.

Prompt:
"Write a job description for [JOB TITLE] at [COMPANY TYPE]. Include: compelling role summary, 5-7 key responsibilities, required qualifications, preferred qualifications, compensation range, benefits, company culture highlights, and growth opportunities. Use inclusive language and focus on impact over tasks."

Performance Review Framework

HR

Design structured performance evaluations that drive employee development.

Prompt:
"Create a performance review template for [ROLE LEVEL] employees. Include: goal achievement assessment, core competencies evaluation, 360-degree feedback section, development planning, and career progression discussion. Provide rating scales, example questions, and manager guidance notes."

Customer Issue Resolution

Support

Generate empathetic and solution-focused responses to customer complaints.

Prompt:
"Create a response template for customer complaint about [ISSUE TYPE]. Include: acknowledgment of frustration, apology, explanation of root cause, detailed resolution steps, prevention measures, compensation offer (if applicable), and follow-up plan. Tone: empathetic but professional."

Knowledge Base Articles

Support

Write clear, searchable help articles that reduce support tickets.

Prompt:
"Write a knowledge base article for [FEATURE/PROCESS]. Include: clear title, problem statement, step-by-step solution with screenshots, troubleshooting section, related articles, and search keywords. Use simple language, bullet points, and numbered lists. Target beginner to intermediate users."

Data Analysis Report

Analytics

Transform raw data into actionable business insights and recommendations.

Prompt:
"Analyze [DATASET/METRICS] and create an executive summary. Include: key findings, trend analysis, performance vs. benchmarks, correlation insights, anomaly identification, business impact assessment, and 3-5 actionable recommendations. Use data visualization suggestions and include confidence levels."
